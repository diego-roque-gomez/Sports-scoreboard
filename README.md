# Sportanzeigetafel
Entwicklung eines funktionsfähigen Prototyps einer Sportanzeigetafel

Dieses Projekt wurde in Zusammenarbeit mit der Sportabteilung des Tecnológico de Monterrey entwickelt und konzentrierte sich auf die Konstruktion einer elektronischen Anzeigetafel für das American-Football-Stadion der Universität.

## Anforderungen ##
- Hohe Sichtbarkeit unter Außenbeleuchtungsbedingungen
- Wetterbeständige Struktur
- Sichere Befestigung in der Stadionkontrollkabine
- Fernbedienbarkeit

## Systemübersicht ##

Ein funktionsfähiger Prototyp wurde entworfen, hergestellt und dem Sportdirektor des Campus zur Bewertung vorgelegt.

Das System implementierte eine IoT-basierte Architektur, in der ein ESP32-Mikrocontroller drahtlose Befehle von einer Webschnittstelle empfing und so die Fernsteuerung der Anzeigetafel ermöglichte.

## Technische Beiträge ##
- Entwurf einer kundenspezifischen Leiterplatte mit mehreren 7-Segment-Anzeigen und 8x8-LED-Matrizen
- Implementierung von MAX7219-Treibern für eine effiziente Anzeigesteuerung und GPIO-Optimierung
- Hardware-Integration mit ESP32
- Mitwirkung am Entwurf eines einziehbaren mechanischen Mechanismus für den Einsatz und den Umweltschutz
